# Vanilla JS Tip Calculator 

A tip calculator that evaluates a tip based on the bill amount, chosen %, and # of people. Results display the tip-per-person and the total-per-person. 

## Features 
* Simple math
* Custom tip %
* Reset button
* Active states
* USD number formatting
* CSS grid + flexbox
